# Project Structure

This document outlines the file and directory structure of the Pretende website.

## Root Directory

The root directory contains the main configuration files and top-level directories for the project.

-   `.gitignore`: Specifies intentionally untracked files to ignore.
-   `manifest.json`: Web application manifest for PWA capabilities.
    -   **Name**: Pretende
    -   **Description**: Producer of illuminated advertising letters
-   `pretende-landing`: Likely a file or directory related to the landing page.
-   `_next/`: Contains the output of the Next.js build. This is where the compiled and optimized assets are stored.
-   `61f851e659d331fc33952ad4/`: Contains image assets.
-   `de/`: Contains the German version of the website pages.
-   `fonts/`: Contains custom fonts used in the project.
-   `images/`: Contains various images used throughout the site.
-   `img/`: Contains additional image assets.
-   `mockup-data/`: Contains data for mockups.

## German Site (`de/`)

This directory holds all the pages for the German version of the website.

-   `index.html`: The main landing page.
-   `angebot/`: Contains pages related to offers and products.
-   `datenschutzpolitik/`: Privacy policy page.
-   `einloggen/`: Login page.
-   `galerie/`: Gallery page.
-   `hilfe/`: Help or FAQ page.
-   `investition/`: Investment-related pages.
-   `karriere/`: Career opportunities page.
-   `kontakt/`: Contact page.
-   `kontakt-form/`: Contact form page.
-   `konzeption/`: Conception or design process page.
-   `korb/`: Shopping cart page.
-   `laden/`: Shop or store page.
-   `ordnung/`: Order or regulations page.
-   `produktion/`: Production process page.
-   `webblog/`: Contains blog posts.
